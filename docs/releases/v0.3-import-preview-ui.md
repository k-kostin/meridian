# v0.3 Import Preview UI

Purpose: add a safe UI layer around the Excel nomenclature import parser.

## What Changed

- Added `/items/import/` for Excel nomenclature import preview.
- Added an upload form for `.xlsx` workbooks.
- Reused `warehouse_app/imports.py` to show parsed preview rows.
- Displayed row-level validation errors before any import commit.
- Added a manager-only CTA from the nomenclature page.
- Restricted the screen to users who can manage references.

## Deferred

- Database commit flow for imported nomenclature.
- Unit auto-creation or mapping.
- Duplicate handling policy.
- Opening balance import through inventory documents.

## Do Not Regress

- Preview must not create or update `Item` records.
- Preview must not create stock documents, inventory documents, or balances.
- View-level permissions must continue to block `viewer` users from import preview.
