# v0.4 Import Column Aliases

Date: 2026-06-04

## Purpose

Make Excel import less brittle by accepting common column aliases.

## Changes

- Item import accepts `SKU`, `–Э–∞–Ј–≤–∞–љ–Є–µ`, and `–Х–і.–Є–Ј–Љ.` style headers.
- Opening inventory import accepts `–Ъ–Њ–і —Б–Ї–ї–∞–і–∞`, `SKU`, and `–Ю—Б—В–∞—В–Њ–Ї` style headers.
- Import still uses preview and validation before any data is committed.

## Do Not Regress

- Imports must still report row-level validation errors before commit.
- Opening stock import must still create a draft full inventory, not direct stock rows.
