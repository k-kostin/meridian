# v0.3 Import Commit Flow

Purpose: allow reference managers to create new nomenclature items from a validated Excel import preview.

## What Changed

- Added validation against existing `Item.sku` values.
- Added validation for unknown `Unit.code` values.
- Added duplicate SKU detection inside the uploaded workbook.
- Added commit action on the import preview screen.
- Created `Item` records only when the full workbook has zero errors.
- Redirected to the nomenclature list after successful import.

## Deferred

- Updating existing items by SKU.
- Auto-creating units.
- Column mapping.
- Opening balance import through inventory documents.

## Do Not Regress

- Import commit must not create stock documents, inventory documents, balances, or stock lines.
- Import commit must not write partial data when any row has an error.
- Import commit must remain restricted to users who can manage references.
