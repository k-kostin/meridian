# v0.4 Category Report Filters

Date: 2026-06-05

## Purpose

Allow operational reports to focus on a selected item category.

## Changes

- Added category filters to balances, daily ledger, monthly ledger, and period analytics.
- Preserved category filters in Excel exports.
- Added category metadata to filtered report workbooks.

## Notes

The filter is applied before report aggregation, including zero-balance rows.
