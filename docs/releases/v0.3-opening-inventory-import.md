# v0.3 Opening Inventory Import

Purpose: create a safe onboarding path for opening stock quantities from Excel.

## Added

- Excel preview for opening stock quantities.
- Validation for warehouse codes, SKUs, duplicate rows, and non-negative numeric quantities.
- Commit flow that creates a draft full inventory document for one warehouse.

## Deliberately Not Included

- Direct balance writes.
- Automatic posting during import.
- Multi-warehouse import in one file.
- Import of unknown nomenclature.

## Invariants

- Opening stock import must preserve calculated-balance semantics.
- The user must review and post the created inventory manually.
- Posting still uses existing inventory adjustment logic.
