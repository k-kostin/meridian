# v0.4 Item Import Unit Auto-Create

Date: 2026-06-05

## Purpose

Allow admins to create missing units during item import through an explicit option.

## Changes

- Item import still rejects unknown units by default.
- A separate checkbox can create missing units during commit.
- New units use the imported unit code as both code and name.
- Duplicate missing unit codes in one file are created once.

## Do Not Regress

- Unit auto-create must stay opt-in.
- Existing units must not be overwritten by import.
- Item import must still validate rows before commit.
