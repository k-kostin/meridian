# v0.2 Activity Timeline

Date: 2026-06-03

## Purpose

Make posted warehouse operations easier to inspect during pilot usage.

This slice adds a small operational timeline to document and inventory detail pages. It is not a full security audit log and does not add user roles or authentication.

## Changes

- Added `ActivityEvent` for stock-affecting events.
- Recorded posted stock documents once.
- Recorded posted inventories once.
- Recorded automatic adjustment creation from inventory.
- Rendered timeline events on document and inventory detail pages.
