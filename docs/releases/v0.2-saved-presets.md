# v0.2 Built-In Filter Presets

Date: 2026-06-04

## Purpose

Make the most common list views faster to reach during daily use.

This slice adds built-in quick filters to movement documents and current balances. It does not add user-defined saved views, persistence, sharing, roles, or database-backed preferences.

## Changes

- Added movement document presets for drafts, posted documents, receipts, issues, and transfers.
- Added current balance presets for by-warehouse, consolidated, all positions, and nonzero-only views.
- Kept explicit form filters higher priority than preset defaults.
- Added chip-style quick filters to the documents and balances screens.

## Deferred Scope

Full saved views remain a future feature. A later version can add named user-defined presets, persistence, ownership, sharing, and default view selection after authentication and roles are introduced.

## Do Not Regress

This version must preserve the `v0.1 MVP Baseline` behavior documented in `docs/releases/v0.1-mvp-baseline.md`.
