# v0.4 User Saved Views

Date: 2026-06-05

## Purpose

Let authenticated users save frequently used document and balance filters.

## Changes

- Added per-user saved views for documents and current balances.
- Added saved-view chips on supported list screens.
- Added delete controls for saved views owned by the current user.

## Notes

Anonymous demo/local users keep the existing built-in presets and cannot save custom views.
