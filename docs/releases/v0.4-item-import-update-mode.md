# v0.4 Item Import Update Mode

Date: 2026-06-04

## Purpose

Allow admins to update existing item cards from Excel through an explicit import mode.

## Changes

- Item import keeps create-only behavior by default.
- A separate update mode changes existing item name, unit, active flag, and notes.
- Update mode does not create new item SKUs.

## Do Not Regress

- Create-only import must still reject existing SKUs.
- Every import action must still pass preview and row-level validation before commit.
- Import modes must not change stock balances directly.
